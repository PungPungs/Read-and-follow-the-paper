#### 텐서의 종류
1. 스칼라 : 0차원 텐서
- [1].[2] 등 방향이 없는 백터
2. 벡터 : 1차원 텐서
- [1,2], [3,4] 등 크기, 방향을 나타내는 벡터
3. 행렬 : 2 차원 텐서
- [[1,2,3],[4,5,6]] 열과 행으로 이루어진 벡터
4. 텐서 : n차원 텐서
- 행과 열을 가진 다차원의 백터


#### Norm

1. L1 Norm
- 백터의 원소 절댓값을 다 더한 값이다.

    $T = [1,2,3,4] -> L1 = |1| + |2| + |3| + |4| = 10$

2. L2 Norm
- 백터의 원소를 제곱 후 다 더한 값을 제곱근을 취하는 방식이다.

    $T= [1,2,3,4] -> L2 = \sqrt{1^2 + 2^2 + 3^2 + 4^2}$
